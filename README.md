# AzureSpace

## 本项目基于vue-cli3脚手架创建-此仓库为项目的前端部分,后端部分在仓库中的azureSpace-backEnd

## 运行前安装依赖

```
npm install
```

### 启动方式

```
npm run serve
```

### 后端接口: 腾讯新闻以及自己编写的一些后端接口,实现博客文章以及登录注册



### 本项目技术栈

**vue**

**vuex**

**vue-router** 

**axios**

**vue-cli3**

**less** 

**element-ui**

**webpack**

**vue-awesome-swiper**

**moment.js**
